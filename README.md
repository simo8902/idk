### Prerequisites  
Make sure you have the following installed:  
- CMake from 3.27 to 3.30.5 max supported
- A C++ compiler (GCC, Clang, LLVM) 
- Git (to clone the repo)

### Steps to Compile  

``` 
mkdir build
cd build
cmake -B build
cmake --build build
```
or using the not ready Makefile
```
make
```
